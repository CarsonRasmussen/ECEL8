## Lab Questions

### In plain English describe the function and use of a multiplexer.
A mux is a circuit that selects one of the many input signals and
sends it to the single output. It's useful when you have many
inputs that share a single output.

### In plain English describe the function and use of a demultiplexer.
A demux is the same thing as a mux but does the exact opposite. It takes
a single input signal and sends it to one of many outputs. It's useful
if you have an input signal that you want to send to various outputs.

### What other uses might these circuits have? (Think Shannon’s)
They could be used to implement boolean functions. Using mux/demux you can
break down complex logic functions into smaller functions to help reduce complexity.
